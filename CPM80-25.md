# CPM80-25 Cycloidal Robotic Joint Module

![CPM80-25](images/cpm80-25-main.jpg)

## Overview

CPM80-25 is a compact cycloidal robotic joint module designed for robotic joints requiring low backlash and high torque density.

The module integrates:

* Cycloidal Reducer
* BLDC Motor
* Encoder Interface
* Driver Integration Option

---

## Key Specifications

| Parameter         | Value              |
| ----------------- | ------------------ |
| Transmission Type | Cycloidal Pinwheel |
| Diameter          | 80 mm              |
| Thickness         | 29.7 mm            |
| Rated Torque      | 10 Nm              |
| Peak Torque       | 50 Nm              |
| Rated Speed       | 120 rpm            |
| Motor Power       | 500 W              |
| Voltage           | 24-48V             |
