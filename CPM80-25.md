# CPM80-25 Compact Cycloidal Robot Joint Actuator Module

## Product Overview

CPM80-25 is a compact cycloidal robot joint actuator module designed for robotic applications requiring low backlash, high torque density, impact resistance, and compact installation.

The module can be configured according to project requirements, including reducer, motor, encoder, driver, and communication options.

## Key Features

- Compact cycloidal transmission
- High torque density
- Low backlash
- Shock-resistant structure
- Flexible motor and control configuration
