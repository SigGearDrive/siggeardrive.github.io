# CPM100-25 Cycloidal Robotic Joint Module

![CPM100-25 Cycloidal Robotic Joint Module](assets/images/CPM100-25.png)

## Overview

CPM100-25 is a high-torque cycloidal robotic joint module optimized for humanoid robot hip joints, knee joints and industrial robotic applications.

Designed specifically for robotics, CPM100-25 offers low backlash, compact dimensions and high torque density.

---

## Key Specifications

| Parameter         | Value     |
| ----------------- | --------- |
| Transmission Type | Cycloidal |
| Diameter          | 100 mm    |
| Thickness         | 29.5 mm   |
| Rated Torque      | 25 Nm     |
| Peak Torque       | 75 Nm     |
| Rated Speed       | 60 rpm    |
| Voltage           | 24-48V    |
| Backlash          | 0.05°     |
