# CPM100-25 Compact Cycloidal Robot Joint Actuator Module

## Product Overview

CPM100-25 is a compact high-torque cycloidal robot joint actuator module designed for robotic systems requiring high torque density, low backlash, shock resistance, and compact installation space.

The configuration can be customized according to project requirements, including motor, encoder, driver, communication interface, and mechanical integration.

## Key Features

- High torque density cycloidal transmission
- Compact axial thickness
- Low backlash
- High rigidity
- Robot joint oriented design
