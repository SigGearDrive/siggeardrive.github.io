## How SigGear Supports Project Evaluation

A typical project evaluation workflow may include:

1. Customer shares application, torque, speed, size, load, and quantity requirements.
2. SigGear reviews whether a standard model can be used.
3. If a standard model is not suitable, SigGear evaluates custom reducer or actuator options.
4. SigGear confirms key technical risks, such as radial load, axial load, duty cycle, backlash, noise, thermal condition, and installation constraints.
5. Customer requests CAD, 2D drawing, datasheet, sample, or quotation.
6. Prototype testing and small-batch evaluation can be discussed according to project requirements.

## Information Needed for Custom Projects

To help SigGear evaluate your project, please provide:

| Required Information | Example |
| --- | --- |
| Application | Humanoid robot, quadruped robot, robot arm, gripper, medical device, AGV / AMR, automation equipment |
