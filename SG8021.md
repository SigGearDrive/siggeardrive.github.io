# SG8021 Planetary Robotic Joint Module

![SG8021](images/sg8021-main.jpg)

## Overview

SG8021 is a high-performance robotic joint actuator designed for humanoid robots, quadruped robots and industrial robotic systems.

The module combines:

* Planetary Gear Transmission
* High Torque BLDC Motor
* Integrated Driver
* Absolute Encoder
* Rich Communication Interfaces

---

## Key Specifications

| Parameter         | Value     |
| ----------------- | --------- |
| Transmission Type | Planetary |
| Diameter          | 100 mm    |
| Thickness         | 42.6 mm   |
| Reduction Ratio   | 9.25:1    |
| Rated Torque      | 10 Nm     |
| Peak Torque       | 30 Nm     |
| Rated Speed       | 160 rpm   |
| Motor Power       | 213 W     |
| Voltage           | 24-48V    |
| Weight            | 671 g     |
