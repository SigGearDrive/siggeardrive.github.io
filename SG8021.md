# SG8021 High-Performance Planetary Robot Joint Actuator

## Product Overview

SG8021 is a compact planetary robot joint actuator for humanoid robots, quadruped robots, robotic arms and industrial robotic systems. The standard configuration combines a planetary transmission, BLDC motor, encoder feedback and an integrated driver solution.

## Key Specifications

| Parameter | Value |
| --- | --- |
| Transmission structure | Planetary |
| Outer diameter | 100 mm |
| Thickness | 42.6 mm |
| Reduction ratio | 9.25:1 |
| Backlash | 15-20 arcmin |
| Rated voltage | 24-48 V |
| Motor power | 213 W |
| Rated speed | 160 rpm |
| No-load speed | 240 rpm |
| Rated torque | 10 Nm |
| Peak torque | 30 Nm |
| Weight | 671 g |
| Allowable radial force | 1300 N |
| Allowable axial force | 550 N |

## Control and Communication

- CAN communication
- MIT control compatibility
- ODrive-compatible interface
- Closed-loop position, velocity and torque control

Driver firmware and low-level control documentation are provided through qualified electronic drive partners according to product configuration.
