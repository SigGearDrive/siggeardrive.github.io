# SG8021 High-Performance Planetary Robot Joint Actuator

## Product Overview

SG8021 is a high-performance planetary robot joint actuator designed for robotic systems requiring compact structure, integrated electronics, high torque density, and fast prototype integration.

The actuator combines:

- Planetary gear transmission
- High-torque BLDC motor
- Integrated driver
- Absolute encoder
- Robotic communication interfaces

## Control and Communication

Supported control features:

- CAN communication
- MIT protocol
- ODrive-compatible interface
- Position control
- Velocity control
- Torque control
- Closed-loop actuator control

CAN protocol documentation is provided after purchase or engineering evaluation.
