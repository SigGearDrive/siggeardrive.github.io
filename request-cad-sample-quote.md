# Request CAD, Datasheet, Sample, or Quote

Use this page to request CAD models, 2D drawings, datasheets, samples, quotations, or custom robotic joint drive solutions from SigGear.

SigGear provides compact precision drive solutions for humanoid robots, quadruped robots, robot arms, exoskeletons, AGV / AMR systems, grippers, medical devices, automation equipment, and custom robotic actuator projects.

For manufacturing capability, quality control focus, and OEM / ODM customization support, see [Factory Capability, Quality Control, and OEM / ODM Support](factory-capability-quality-oem-odm.md).

## What You Can Request

| Request Type | Description |
| --- | --- |
| CAD model | Request CAD reference files for mechanical evaluation. |
| 2D drawing | Request outline drawing, mounting interface drawing, or dimension drawing. |
| Datasheet | Request product datasheet or technical specification confirmation. |
| Sample | Request prototype sample evaluation for robot or automation projects. |
| Quote | Request quotation based on product model, quantity, and customization requirements. |
| Custom solution | Request a customized reducer, motor-matched reducer, or compact robotic joint module. |

## Please Include These Requirements

To help SigGear recommend the right model or prepare a quotation, please include as much of the following information as possible.

| Information | Example |
| --- | --- |
| Your name | John Smith |
| Company name | Robotics startup / automation company / university lab |
| Company website | Website or LinkedIn page |
| Country / region | United States, Germany, Japan, etc. |
| Product of interest | CPM100-25, CPM80-25, SG6010C, SG8021, 6-42mm planetary gearbox, or custom solution |
| Request type | CAD model, 2D drawing, datasheet, sample, quote, or custom solution |
| Application | Humanoid robot, quadruped robot, robot arm, exoskeleton, AGV / AMR, gripper, medical device, automation equipment |
| Required output torque | Rated torque and peak torque |
| Required output speed | Target output speed in rpm |
| Installation space | Diameter, length, thickness, mounting limits, shaft or flange requirement |
| Load condition | Radial load, axial load, moment load, impact load, duty cycle |
| Precision requirement | Backlash, repeatability, stiffness, or positioning requirement |
| Motor / controller requirement | BLDC motor, servo motor, CAN, RS485, EtherCAT, encoder, brake, driver, or external controller |
| Quantity | Prototype quantity and estimated annual quantity |
| Project stage | Concept design, prototype, testing, small batch, or mass production |
| Additional notes | Special environment, noise, lifetime, weight, material, or certification requirements |
