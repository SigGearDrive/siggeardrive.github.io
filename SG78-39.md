# SG78-39 Compact Cycloidal Joint Actuator

## Product Overview

SG78-39 is a compact cycloidal joint module designed for robotic systems requiring high torque density and compact installation.

## Key Specifications

| Parameter | Value |
| --- | --- |
| Transmission | Cycloidal |
| Outer Diameter | 78.7 mm |
| Thickness | 35.8 mm |
| Weight | 600 g |
| Rated Torque | 20 Nm |
| Peak Torque | 52 Nm |
| Reduction Ratio | 39:1 |
| Backlash | 5-10 arcmin |
| Voltage | 36 V |
| Rated Power | 200 W |
| Rated Speed | 48 rpm |
| Sensor | Hall Sensor |

## Sensor and Control

SG78-39 provides Hall sensor feedback.

Unlike integrated robot joint actuators, SG78-39 does not include an integrated driver.
