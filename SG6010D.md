# SG6010D Integrated Planetary Robot Joint Actuator

## Product Overview

SG6010D is an integrated planetary robot joint actuator for humanoid robots, exoskeletons, robotic arms and compact robotic platforms.

## Key Specifications

| Parameter | Value |
| --- | --- |
| Transmission structure | Planetary |
| Outer diameter | 80 mm |
| Thickness | 58.77 mm |
| Reduction ratio | 28.13:1 |
| Backlash | 15-30 arcmin |
| Rated voltage | 24-48 V |
| Motor power | 328 W |
| Rated speed | 100 rpm |
| No-load speed | 170 rpm |
| Rated torque | 16 Nm |
| Peak torque | 50 Nm |
| Weight | 791 g |
| Allowable radial force | 1000 N |
| Allowable axial force | 400 N |

## Control and Communication

- Integrated driver solution
- Encoder feedback
- CAN communication
- MIT control compatibility
- ODrive-compatible interface
- Closed-loop position, velocity and torque control

Driver firmware and low-level control documentation are provided through qualified electronic drive partners according to product configuration.
