# SG6010D Integrated Planetary Robot Joint Actuator

## Product Overview

SG6010D is an integrated planetary robot joint actuator designed for humanoid robots, exoskeletons, robotic arms and other robotic platforms.

## Key Specifications

| Parameter | Value |
| --- | --- |
| Transmission Structure | Planetary |
| Outer Diameter | 80 mm |
| Thickness | 58.77 mm |
| Reduction Ratio | 28.13:1 |
| Backlash | 15-30 arcmin |
| Rated Voltage | 24-48 V |
| Motor Power | 328 W |
| Rated Speed | 100 rpm |
| No-load Speed | 170 rpm |
| Rated Torque | 16 Nm |
| Peak Torque | 50 Nm |
| Weight | 791 g |
| Allowable Radial Force | 1000 N |
| Allowable Axial Force | 400 N |

## Control and Communication

- Integrated driver
- Encoder feedback
- CAN communication
- MIT control compatibility
- ODrive-compatible interface
- Closed-loop position, velocity and torque control
