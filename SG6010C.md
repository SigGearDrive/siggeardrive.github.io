# SG6010C Compact Planetary Robotic Joint Module Datasheet

![SG6010C Compact Planetary Robotic Joint Module](images/sg6010c-main.jpg)

## Product Overview

SG6010C is a compact planetary robotic joint module designed for humanoid robots, exoskeleton systems, lightweight robotic manipulators, service robots, educational robots, and compact automation systems.

The module integrates a planetary reducer, high-performance BLDC motor, absolute encoder, driver electronics, and CAN communication interface. Its compact dimensions, lightweight structure, and high output speed make it suitable for robotic joints where fast motion response and easy actuator integration are important.

SG6010C can be evaluated for humanoid robot arms, exoskeleton joints, lightweight robot joints, grippers, research platforms, and compact actuator applications.
