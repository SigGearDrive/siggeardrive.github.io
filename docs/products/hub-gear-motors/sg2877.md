# SG-2877 3 Nm Hub Gear Motor

## Product Overview

SG-2877 is a compact 99 mm hub gear motor using a planetary reduction structure. It is designed for AGV, AMR, mobile robots, rehabilitation platforms and compact wheel-drive systems that require moderate torque in a compact package.

The standard catalog configuration uses Hall sensors. An external motor controller is required unless a controller is specifically included in the quotation.
