# SG-8021 Integrated Planetary Robot Joint Actuator

## Product Overview

SG-8021 is a low-profile planetary robot joint actuator designed for humanoid robots, quadruped robots, robotic arms, exoskeletons and compact automation systems.

![SG-8021 integrated planetary robot joint actuator front-angle view](../../assets/images/products/robot-joint-actuators/sg8021/sg8021-robot-joint-actuator-front-angle.webp){ width="720" loading="lazy" }

Available configurations:

- Motor + planetary gearbox + integrated driver + encoder
- Motor + planetary gearbox without integrated driver, with the final sensor and cable configuration confirmed by order

## Key Benefits

- 100 mm outer diameter with a 42.6 mm low-profile body
- 10 Nm rated torque and 30 Nm peak torque
- 160 rpm rated output speed
- 15-20 arcmin backlash
- Optional integrated driver and absolute-encoder control platform

## Encoder and Sensor

The driver-equipped configuration uses the SigGear integrated driver and absolute-encoder control platform. Final encoder, sensor and cable configuration must be confirmed in the quotation and order documentation.

## Driver and Control

For the driver-equipped version, available functions include:

- FOC closed-loop control
- Position control
- Velocity control
- Torque control
- MIT-style motion control
- PID parameter configuration
- Calibration and user-zero configuration

These functions apply only to the driver-equipped configuration.

## Communication Interfaces

Standard on the driver-equipped configuration:

- CAN
- USB Type-C for debugging and configuration

Optional or project-specific:

- RS485 through expansion hardware
