# SG-6010D Integrated Planetary Robot Joint Actuator

## Product Overview

SG-6010D is an integrated planetary robot joint actuator designed for humanoid robots, quadruped robots, robotic arms and exoskeleton applications.

Available configurations:

- Motor + planetary gearbox + integrated driver + encoder
- Motor + planetary gearbox without integrated driver (configuration confirmed by order)

## Key Specifications

| Parameter | Value |
| --- | --- |
| Transmission structure | Planetary gearbox |
| Outer diameter | 80.0 mm |
| Thickness | 58.77 mm |
| Reduction ratio | 28.13:1 |
| Backlash | 15-30 arcmin |
| Operating voltage | 24-48 VDC |
| Motor power | 328 W |
| Rated speed | 100 rpm |
| No-load speed | 170 rpm |
| Rated torque | 16 Nm |
| Peak torque | 50 Nm |
| Weight | 791 g |
| Allow radial force | 1000 N |
| Allow axial force | 400 N |

## Peak Torque Condition

Peak torque: 50 Nm

Peak duration: Subject to operating voltage, current limit and thermal conditions.

## Encoder and Driver

Driver-equipped version supports:

- MA600 16-bit single-turn absolute magnetic encoder
- FOC closed-loop control
- CAN communication
- USB Type-C debugging
- Position, velocity, torque and MIT-style motion control

## Communication

Standard:

- CAN
- USB Type-C

Optional/project confirmation:

- RS485
- EtherCAT
- Other expansion interfaces
