# CPM-78-39 Integrated Cycloidal Joint Module

## Product Overview

CPM-78-39 is a compact cycloidal robot joint module designed for applications that require high torque, low backlash and a short axial package.

The standard catalog configuration uses Hall sensors and does not include an integrated driver or absolute encoder. Driver, communication and closed-loop control functions must not be assumed unless they are separately confirmed in the quotation and technical agreement.

## Sensor and Driver Configuration

The standard CPM-78-39 catalog configuration uses Hall sensors.

The following features are **not** included as standard claims for this model:

- Integrated driver
- MA600 absolute encoder
- CAN communication
- RS485 communication
- EtherCAT communication
- Position, velocity or torque closed-loop control

Any external driver, encoder, communication interface or control solution must be evaluated and confirmed as a separate project configuration.
