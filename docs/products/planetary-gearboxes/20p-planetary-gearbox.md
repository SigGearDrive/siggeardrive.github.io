# 20P 20 mm Micro Planetary Gearbox

## Product Overview

The SigGear 20P is a 20 mm micro planetary gearbox designed for compact robotic mechanisms, precision motion systems and small automated equipment.

## Product Identity

| Item | Description |
| --- | --- |
| Model | 20P |
| Product type | Micro planetary gearbox |
| Nominal outer diameter | 20 mm |
| Number of stages | 1 to 4 |
| Housing material | Steel |
| Gearwheel material | Steel |
| Output bearing | Ball bearing |

| Parameter | Published value |
