## Model Naming Rules

- Preserve official capitalization and punctuation: `SG-6010D`, not `SG6010D`; `CPM-80-25`, not `CPM80-25`.
- Do not create a new model alias solely for SEO or sales convenience.
- If an older catalog uses a different model name, record the relationship only after engineering or product management confirms it.
- A customer-specific configuration must be identified in the quotation or technical agreement and must not silently replace the standard public page.

## Missing or Conflicting Model Data

If a model is missing, a link is incorrect or two sources disagree, use the repository's Product Data Update issue template. Include the source document, revision or date, affected model and exact field requiring review.
