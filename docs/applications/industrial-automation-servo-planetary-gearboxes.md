# Servo Motor Planetary Gearboxes for Industrial Automation

## Application Overview

Industrial automation equipment often uses servo motors with planetary gearboxes to increase output torque, reduce motor speed, improve inertia matching and create compact motion axes. Typical applications include pick-and-place systems, packaging machines, rotary indexing tables, conveyor adjustment axes, compact linear actuators, inspection equipment, fixtures, grippers and small machine mechanisms.
