# SG6010D Integrated Planetary Robot Joint Actuator

## Product Overview

SG6010D is an integrated planetary robot joint actuator designed for humanoid robots, exoskeletons, robotic arms and compact robotic systems.

The actuator integrates:

- Planetary reducer
- BLDC motor
- Absolute magnetic encoder
- Integrated driver
- Communication interface

## Key Specifications

| Parameter | Value |
| --- | --- |
| Transmission | Planetary |
| Outer Diameter | 80 mm |
| Rated Torque | 16 Nm |
| Peak Torque | 50 Nm |
| Rated Speed | 100 rpm |
| Voltage | 24-48 V |
| Weight | 791 g |
| Backlash | 15-30 arcmin |
| Encoder | Absolute magnetic encoder |
| Resolution | 14-bit |
| Encoder Interface | SPI / UART |

## Control and Communication

Supported:

- CAN communication
- RS485
- PWM
- MIT control
- ODrive compatible interface
