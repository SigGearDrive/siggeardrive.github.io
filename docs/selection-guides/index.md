# Selection Guides

SigGear selection guides help robotics and automation engineers convert application requirements into a practical motor, gearbox or joint-actuator shortlist.

The guides are intended for preliminary engineering selection. Final model approval requires review of the complete torque-speed cycle, installation structure, external loads, thermal conditions, feedback architecture, controller limits and project configuration.

## Published Guides

### Robot Joint Actuator Selection Guide

A step-by-step guide for humanoid robots, quadruped robots, robotic arms, exoskeletons and other rotary joints. It covers load definition, torque estimation, output speed, duty cycle, transmission choice, external loads, positioning, feedback, driver and communication requirements.

## Information to Prepare Before Selection

For faster review, prepare:

- Application and axis position
- Payload, link length and center of gravity
- Continuous torque
- Peak torque and peak duration
- Rated and maximum output speed
- Acceleration and deceleration profile
- Operating voltage
- Maximum outer diameter, thickness and weight
- Radial, axial and overturning loads
- Backlash, positioning and repeatability requirements
- Duty cycle, ambient temperature and cooling method
- Encoder, driver and communication requirements
- Brake and power-off behavior
- Prototype quantity and annual forecast

Product specifications, drawings and control functions apply only to the selected model and configuration confirmed during engineering review and quotation.
