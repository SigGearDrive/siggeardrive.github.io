# SigGear Product Documentation 2.0

This branch contains the clean rebuild of the SigGear product documentation website.

## Documentation rules

- Product pages are the primary source of truth.
- Application pages, selection guides, comparisons, and developer resources must reference approved product pages.
- Unverified specifications must be marked as `Pending engineering confirmation`.
- Do not duplicate product pages or maintain separate copies outside `docs/`.
- Use lowercase file names and hyphen-separated paths.
- Use only the official contact email below
