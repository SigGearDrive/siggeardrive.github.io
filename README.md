# SigGear Product Documentation

SigGear provides compact precision drive solutions for robotics, automation, medical devices, laboratory equipment, compact actuators, and intelligent motion systems.

This documentation site helps engineers evaluate SigGear compact cycloidal robotic joint modules, 6-42mm planetary gear reducers, micro gear motors, and customized drive solutions for robot joints, humanoid robots, quadruped robots, robot arms, robotic grippers, AGV / AMR systems, medical devices, and precision automation equipment.
