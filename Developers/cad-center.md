# SigGear CAD Center

SigGear provides mechanical reference resources for engineers developing robot joint actuators, robotic systems, and compact automation equipment.

## CAD Resource Strategy

To support engineering evaluation while protecting project-specific design data:

### Public Resources

Available for initial evaluation:

- Product images
- Basic dimensions
- Product specifications
- Mechanical reference information

### Request Access

Available after engineering evaluation or project discussion:

- STEP models
- Integration CAD files
- Product-specific mechanical drawings

### Project Cooperation Resources

Provided based on project requirements:

- Detailed engineering drawings
- Customized mounting interfaces
- Application-specific CAD support
