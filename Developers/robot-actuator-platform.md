# SigGear Robot Joint Actuator Platform

SigGear provides integrated robot joint actuator solutions for engineers developing humanoid robots, quadruped robots, robotic arms, service robots, and compact automation systems.

Unlike a traditional gearbox supplier, SigGear focuses on actuator-level integration, combining:

- Precision reducers
- BLDC motor integration
- Encoder feedback options
- Integrated control solutions
- Mechanical customization support

## Platform Architecture

### Mechanical Layer

- Planetary robot joint actuators
- Cycloidal robot joint modules
- Compact planetary reducers
- Custom gearbox solutions

### Feedback Layer

Supported configurations may include:

- Magnetic encoders
- Absolute encoders
- Position feedback integration

### Control Integration

Selected actuator configurations support:

- CAN communication
- MIT control compatibility
- ODrive ecosystem compatibility
- Fault diagnostics
- Firmware update support

## Driver and Control Interface Notice

Integrated driver solutions used in selected SigGear actuator products are provided through qualified electronic drive partners.

Low-level firmware, driver software, and detailed communication documentation are provided according to product configuration and engineering requirements.
